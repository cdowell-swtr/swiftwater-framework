# swiftwater-framework

A CLI scaffold framework that generates solid, observable, testable Python applications from the ground up — with TDD, quality gates, observability, and environment parity built in from the first line of code.

---

## Who is this for?

**Evaluating the framework?**
Start with the Overview: [Why this framework](overview/why.md), [What you get](overview/what-you-get.md), and the [Quickstart](overview/quickstart.md). Everything you need to decide is on those pages — no repo clone required.

**Building with the framework?**
You scaffolded a project with `framework new`. Your generated project ships its own `CLAUDE.md` and `Taskfile.yml` that cover the day-to-day. The [CLI reference](reference/cli.md) covers every command for updating and managing your project.

**Working inside a generated project?**
The framework's Claude Code integration (`CLAUDE.md` + `.claude/settings.json`) is your primary guide. The [CLI reference](reference/cli.md) covers `framework integrity`, `framework upskill`, and the review commands.
